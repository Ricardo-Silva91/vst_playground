Spatial Effect
The swell that arrives before the sound.

Reverse Reverb captures audio in short windows, applies reverb, strips the dry signal, reverses the tail, and plays it back timed to land just before each transient. The result: a ghostly bloom that swells into every note or hit, as if the room is breathing in anticipation. Window size controls how far ahead the effect reaches — and doubles as the plugin's latency.

macOS Windows
Signal Chain
Input› Capture Buffer› Reverb (wet only)› Reverse tail› Wet Mix› Output

Audio is continuously captured into a rolling buffer of length defined by Window ms. That buffer is fed through a reverb engine with the dry signal subtracted — wet tail only. The tail is reversed, then blended with the dry signal: Dry × (1−Wet) + Reversed tail × Wet. Because the reversed audio plays out before the dry signal catches up, the bloom precedes every attack. Most DAWs compensate for the latency automatically via Plugin Delay Compensation.

Parameters
Room Size

Size of the reverb space. Larger rooms produce longer, more diffuse tails. Smaller values create tight, intimate blooms. Larger values give vast, cathedral-like swells.

Range: 0.1 → 1.0    Default: 0.8
Wet Mix

Blend between dry signal and reversed reverb tail. At low values the effect is subliminal. At full wet, the original signal disappears entirely into the swell.

Range: 0.0 → 1.0    Default: 0.8
Window ms

Audio capture window length. Longer windows produce bigger blooms. Window size determines plugin latency exactly — a 500 ms window introduces 500 ms of latency. Short windows stay rhythmically tight; long windows build massive, slow swells.

Range: 100 → 2000 ms    Default: 500 ms
Presets
Default

Balanced starting point. Room 0.8 / Wet 0.8 / Window 500 ms.

Ghost Bloom

Large, dreamy swell — best on pads and vocals where the build-up has time to unfold. Room 0.95 / Wet 0.9 / Window 800 ms.

Subtle Breath

Gentle and transparent. Good on acoustic sources where you want dimension without drama. Room 0.5 / Wet 0.4 / Window 300 ms.

Wash Out

Fully submerged — the dry signal is entirely replaced by the reversed bloom. Best as a transition effect or texture layer. Room 1.0 / Wet 1.0 / Window 1500 ms.

Tight Shimmer

Short window keeps the bloom punchy and rhythmically coherent. Works on drums and percussion where a longer swell would smear the groove. Room 0.6 / Wet 0.7 / Window 150 ms.

Use Cases
Ghostly pre-echo on vocals

The reversed bloom reinforces each phrase entry with a surreal anticipatory swell — a whisper of the note before it arrives. Keep wet mix low for subliminal depth, or automate it up during held notes for a full-blown effect.

Room Size: 0.6 / Wet Mix: 0.2–0.4 / Window: 200–400 ms
Snare blooms and drum buildups

A short window creates a tension-building rise before the impact. Keep wet mix subtle so the attack still hits hard — the classic film score technique for "breathe before the slam."

Room Size: 0.5 / Wet Mix: 0.2–0.35 / Window: 100–200 ms
Atmospheric pads

On a sustained pad the reversed bloom reinforces each harmonic change with a ghostly swell, adding movement without rhythm. Works beautifully in parallel with a standard reverb.

Room Size: 0.7+ / Wet Mix: 0.4–0.6 / Window: 300–800 ms
Cinematic sound design

Automate Wet Mix up during silence gaps. The reversed tail fades out the preceding phrase backwards, creating a surreal dissolve into the next section. At full wet with a long window the plugin becomes a texture generator.

Room Size: 0.6 / Wet Mix: automate 0 → 0.8 / Window: 500–1500 ms
Tips and Workflow
▶

Run it in parallel. Keep the dry signal intact on a separate track and blend the fully wet plugin return to taste. This avoids pumping at the attack.

▶

Match window to tempo. At 120 BPM, one beat is 500 ms. Setting Window ms to 250 ms gives a rhythmically locked bloom that anticipates every other hit without smearing the groove.

▶

Check your DAW's Plugin Delay Compensation. The bloom should arrive just before the dry transient, not after it. If timing feels off, confirm PDC is enabled in your DAW's settings.

▶

Chain into a forward reverb. Route Reverse Reverb into a regular room reverb on a send. The result is a bloom into the hit, then a natural decay out of it — a complete envelope of space around each sound.

▶

Automate Wet Mix for tension. Gradually raising it over 4–8 bars before a drop builds subliminal anticipation. Pull it back to zero on the downbeat.
